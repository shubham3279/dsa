# INTRODUCTION

## DEFINITION

***Data Structures referes to storage and organization of data, purposed for efficient search, retrieval and manipulation by the algorithm which operates over it.***



## REMARKS


## 3. CLASSIFICATION

* Depending on the organization and access of data, data structures are classified into:

### 3.1 Linear Data Structures

* Data is accessed in a sequntial order, irrespective of how it is organized.
* They are:
    * Arrays
    * Linked Lists
    * Stacks
    * Queues


### 3.2 Non Linear Data Structures

* Data i stored as well as accessed in a non-sequential order.
* They are:
    * Trees
    * Graphs
